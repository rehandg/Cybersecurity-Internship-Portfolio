# TASK_7-Browser-Extentions
Identify and Remove Suspicious Browser Extensions

## Objective
Learn to spot and remove potentially harmful or unnecessary browser extensions to improve browser performance and security.

## Tools Used
- Google Chrome (you can use any modern browser like Firefox, Edge, etc.)

## Task Description
This task involves inspecting the installed browser extensions, identifying any that seem suspicious or are no longer needed, and removing them. This helps prevent data theft, intrusive ads, and poor browser performance caused by malicious extensions.

## Steps Performed

1. Opened Browser Extensions Page:
   - Accessed `chrome://extensions/` in Google Chrome to view all installed extensions.

2. Reviewed Installed Extensions:
   - Carefully examined the list of extensions.
   - Checked each extension's permissions, developer reputation, and user reviews.

3. Identified Suspicious Extensions:
   - Marked the following extensions as suspicious:
     - Free VPN Proxy: Had overly broad permissions (access to all website data), unknown developer, and low reviews.
     - Shopping Assistant: Redirected users to unknown shopping websites and had many red flags in user reviews.

4. Removed Unnecessary Extensions:
   - Uninstalled the suspicious extensions from the browser.

5. Restarted the Browser:
   - Relaunched Chrome to ensure changes took effect.
   - Noticed improvements in browser speed and responsiveness.

6. Researched Browser Extension Threats:
   - Learned that malicious extensions can:
     - Steal browsing history and personal data.
     - Inject ads or redirect websites.
     - Slow down the browser performance.

7. Documented the Process:
   - Created screenshots of key steps.
   - Saved a written report listing removed extensions.
   - Compiled everything into a ZIP file for submission.
